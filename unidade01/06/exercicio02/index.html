Utilizando a tag <img> para inserir imagem externa

Esta é uma página simples que demonstra o uso da tag <img>, exibindo uma imagem do xicara de café.

Podemos utilzar o atributo src para inserir uma imagem de uma URL externa, como a imagem abaixo, que é de um café.
Busque uma imagem de sua preferência, copie seu endereço na web e utilize-o no atributo src.

Link para acessar as imagens utilizadas nesse exercício: https://unsplash.com/pt-br/s/fotografias/dogs
