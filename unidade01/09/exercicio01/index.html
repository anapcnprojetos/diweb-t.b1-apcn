Trabalhando com CSS incorporado

Estilizando elementos

O CSS incorporado, também conhecido como CSS interno, é uma forma de aplicar estilos diretamente em um documento HTML, utilizando a tag <style> dentro do elemento <head>

Ele permite que você defina regras de estilo para elementos específicos da página, sem a necessidade de criar um arquivo CSS separado.

No entanto, é importante usar o CSS incorporado com moderação, pois ele pode tornar o código HTML mais difícil de manter e reutilizar.
