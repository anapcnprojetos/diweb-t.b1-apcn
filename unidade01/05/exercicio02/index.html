Utilizando a tag para inserir imagens em uma página web

Esta é uma página simples que demonstra o uso da tag <img>, exibindo uma imagem do dia da cerveja.

Abaixo temos a utilização do atributo src.
Ele especifica o caminho da imagem ou fonte da imagem. Depois, temos o atributo alt.
Que fornece um texto alternativo se a imagem não puder ser exibida.

Uma regra importante é sempre fornecer um texto alternativo (alt) para as imagens, para garantir a acessibilidade da página e permitir que os usuários com deficiências visuais possam entender o conteúdo da imagem.
