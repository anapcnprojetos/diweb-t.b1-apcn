Estilizando elementos com CSS Inline

Para utilizar o estilo inline, basta adicionar o atributo style diretamente no elemento HTML que deseja estilizar. Podemos ter uma ou várias propriedades CSS dentro do atributo, separadas por ponto e vírgula.

O estilo inline tem a maior prioridade na hierarquia de estilos, ou seja, ele sobrescreve os estilos definidos em folhas de estilo externas ou internas.
No entanto, é recomendado usar o estilo inline com moderação, pois pode dificultar a manutenção do código e a reutilização de estilos em diferentes elementos.

Algumas propriedades CSS

text-align: Define o alinhamento do texto (left, center, right e justify).
font-size: Define o tamanho da fonte. O tamanho padrão é 16px. Podemos aumentar ou diminuir o tamanho.
font-style: Define o estilo da fonte (normal, italic e oblique).
font-weight: Define o peso da fonte (normal, bold e bolder). Além disso, podemos usar valores numéricos de 100 a 900 para especificar o peso da fonte.
